# Runtime Routing

1. Start with `pi-agent-control`.
2. Pick one target atom:
   - `pi-agent-cli` + `tuistory` for normal Pi Coding Agent CLI TUI capture.
   - `true-input` + `pi-agent-cli` for real terminal keyboard/rendering proof.
   - `agent-browser` for web/Electron flows.
   - `true-input` + `pty-capture` for byte-level keyboard proofs.
3. Always load `capture` and `verify`.
4. Load `compose` when producing a video or annotated artifact.
5. Load `showcase` when producing branded/window-chrome/cinematic output.
