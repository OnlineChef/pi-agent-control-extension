# Pi Agent CLI Control Instructions

This repository is the Pi Agent CLI Control conversion of the uploaded Droid Control package.

Always start routed workflows from `pi-agent-control`. Preserve the capture -> compose -> verify discipline. Do not fake evidence. Prefer real terminal/browser automation, record exact commands, and report REFUTED or INCONCLUSIVE when evidence does not support the claim.

Active skills are available in: .agents/skills, .pi/skills, skills.

Runtime root environment variable: `PI_AGENT_CONTROL_ROOT`.
